J — Alignements (items / self / content)

Dans la cellule : justify/align-items définissent le défaut ; justify/align-self surchargent par item. Dans le container : justify/align-content distribuent les pistes quand il reste de l’espace.

Modifiez dans le CSS : place-items, place-self, justify-content, align-content pour constater les effets.

.align-grid{ display:grid; grid-template-columns:repeat(3,120px); grid-auto-rows:90px; gap:12px; place-items:stretch center; }
.align-grid .cell:nth-child(2){ place-self:start end; }
.align-grid .cell:nth-child(3){ align-self:center; }
.align-grid .cell:nth-child(4){ justify-self:start; }

.align-wrapper{ display:grid; grid-template-columns:repeat(3,140px); grid-template-rows:repeat(2,100px); gap:16px;
