I — grid-auto-flow: column

Auto-placement vertical (haut → bas), création de colonnes implicites à droite. Les items qui "span" plusieurs lignes peuvent laisser des trous.

Comparaison : comportement par défaut vs auto-flow: column

● row (défaut) : remplit ligne par ligne → ● column : remplit colonne par colonne

grid-auto-flow: row (défaut)

1
2
3
4
5
6

grid-auto-flow: column

1
2
3
4
5
6

Observez comment les mêmes éléments se placent différemment : row va de gauche à droite puis ligne suivante, column va de haut en bas puis colonne suivante.

<div class="demo-auto-flow-row">
  <div class="cell">1</div>
  <div class="cell">2</div>
  <div class="cell span-2">3</div>
  <!-- ... -->
</div>
<div class="demo-auto-flow-col">
  <div class="cell">1</div>
  <div class="cell">2</div>
  <div class="cell span-2">3</div>
  <!-- ... -->
</div>
.demo-auto-flow-row{ 
  display: grid; 
  grid-template-columns: repeat(3, 160px); 
  grid-auto-flow: row; /* défaut */
  gap: 12px; 
}
.demo-auto-flow-col{ 
  display: grid; 
  grid-template-rows: repeat(3, 80px); 
  grid-auto-flow: column; 
  grid-auto-columns: 160px; 
  gap: 12px; 
}
.span-2{ grid-row: span 2; }
