H — auto-fit vs auto-fill

auto-fit compacte les "pistes vides" ; auto-fill garde le gabarit de pistes, même si elles sont vides.

● auto-fit : compacte l'espace | ● auto-fill : préserve les colonnes vides

Redimensionnez la fenêtre pour voir comment auto-fit étend les cartes vs auto-fill qui garde l'espace pour les colonnes vides.

  <div class="card">auto-fit 1</div>
  <div class="card">auto-fit 2</div>
</section>
<section class="gallery fill">
  <div class="card">auto-fill 1</div>

.gallery{ 
  display: grid; 
  gap: 12px; 
  flex: 1 1 320px;

.gallery.fit{

}

  grid-template-columns: repeat(auto-fill, minmax(140px,1fr));
