G — grid-template-areas

Disposition déclarative : donnez des noms aux zones pour exprimer visuellement le layout.

● header | ● nav | ● main | ● aside | ● footer

En‑tête
Navigation
Contenu principal
Aside
Pied de page

En mobile, redéclarez grid-template-areas en une seule colonne pour empiler les zones.

<div class="demo demo-areas">
  <header>En‑tête</header>
  <nav>Navigation</nav>
  <main>Contenu principal</main>
  <aside>Aside</aside>
  <footer>Pied de page</footer>
</div>
.demo-areas{ 
  display: grid; 
  grid-template-columns: 1fr 3fr 1fr; 
  grid-template-rows: auto 1fr auto; 
  gap: 12px;
  grid-template-areas: 
    "header header header" 
    "nav main aside" 
    "footer footer footer"; 
}
.demo-areas header{ grid-area: header; background: #dff7e8; border: 1px solid #b7e7c8; }
.demo-areas nav{ grid-area: nav; background: #e8e8ff; border: 1px solid #cacaFF; }
.demo-areas main{ grid-area: main; background: #fffbe6; border: 1px solid #f2e2a2; }
.demo-areas aside{ grid-area: aside; background: #fdeef6; border: 1px solid #f3c0dc; }
.demo-areas footer{ grid-area: footer; background: #e6f2ff; border: 1px solid #bfd8ff; }
